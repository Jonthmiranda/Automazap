# Automazap

## Description

Automation system in Python for sending menus and descriptions to contacts on Whatsapp.

libraries: Selenium, Pyautogui, Pandas and urllib.

## Installation

```bash
git clone https://github.com/Jonthmiranda/Automazap
```
